I've recently started diving into self-hosting and building out a home lab.

This started with NixOS on my personal laptop (a ThinkPad T480) and has expanded to a
few other old boxes.

I have two services so far:

1. A git [forge](https://git.desertthunder.dev/) running forgejo (like [Codeberg](https://codeberg.org/)).
2. A [Tangled](https://tangled.org/) knot, to host repos shared on Tangled.

The knot is the small, headless Git server side of that setup.

You can learn more about my setups by looking at the [book](https://nix.desertthunder.dev)
or at the [codebase](https://git.desertthunder.dev/desertthunder/dotfiles)
