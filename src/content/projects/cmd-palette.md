A highly customizable command palette component for React that brings the power of VS Code's command palette to any web application. Built with accessibility and performance in mind.

## Features

- **Keyboard Navigation**: Full keyboard support with customizable shortcuts
- **Fuzzy Search**: Built-in fuzzy matching for command discovery
- **Theming**: Full theming support via CSS variables
- **Accessible**: WCAG 2.1 AA compliant with full screen reader support
