Terminus is a high-performance terminal emulator that runs in the browser using Rust compiled to WebAssembly. It leverages WebGL for GPU-accelerated rendering to achieve smooth scrolling and low latency input handling.

## Key Features

- **WebAssembly Core**: Rust-based terminal emulation engine
- **WebGL Rendering**: GPU-accelerated text rendering for 60fps performance
- **Ligature Support**: Full font ligature rendering for programming fonts
- **Custom Protocol**: Efficient binary protocol for terminal communication
